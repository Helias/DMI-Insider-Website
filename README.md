Prima di tutto clona il progetto:
```
git clone https://github.com/Herbrant/DMI-Insider-Website.git
```
Per lanciare il server di Hugo:
```
cd DMI-Insider-Website
hugo server -D
```

Troverai il sito web su http://localhost:1313
